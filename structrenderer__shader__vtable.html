Parameters
[in]shader_type_シェーダー種別指定値
[in]shader_source_シェーダーソース
[in,out]shader_handle_シェーダー関連リソース管理構造体インスタンスへのポインタ
Return values
RENDERER_INVALID_ARGUMENT以下のいずれか
  • shader_source_ == NULL
  • shader_handle_ == NULL

  • 指定したシェーダー種別はすでにコンパイル済み
  • シェーダープログラムがすでにリンク済み
  • メモリシステム未初期化
RENDERER_SHADER_COMPILE_ERRORシェーダーソースコンパイルエラー
RENDERER_LIMIT_EXCEEDEDメモリシステム使用可能範囲上限超過
RENDERER_NO_MEMORYメモリ確保失敗
RENDERER_SUCCESS処理に成功し、正常終了

pfn_renderer_shader_destroy renderer_shader_vtable::renderer_shader_destroy

シェーダーGPUリソース内部状態管理構造体が管理するリソースを破棄し、自身のメモリも解放する。メモリ解放後はNULLで初期化する

Note
以下のGPUリソースを破棄する
  • バーテックスシェーダー
  • フラグメントシェーダー
  • OpenGLシェーダープログラム
2重destroyを許可する
Parameters
