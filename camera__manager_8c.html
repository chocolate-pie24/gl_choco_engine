camera_result_t camera_manager_camera_get ( int16_t  camera_id_,
const camera_manager_t *  camera_manager_,
camera_t **  out_camera_ 
)

カメラIDを使用して対応するカメラ管理システムからカメラ構造体インスタンスへのポインタを取得する

Parameters
[in]camera_id_取得対象カメラ識別子
[in]camera_manager_カメラ管理システム構造体インスタンスへのポインタ
[out]out_camera_カメラ構造体インスタンスへのポインタ格納先
Return values
CAMERA_INVALID_ARGUMENT以下のいずれか
  • camera_manager_ == NULL
  • camera_manager_->max_camera_count <= camera_id_
  • out_camera_ == NULL
  • camera_id_ < 0
CAMERA_BAD_OPERATION以下のいずれか
  • camera_manager_->max_camera_count <= 0
  • camera_manager_->camera_array == NULL
  • camera_id_に対応するカメラが管理システム内に見つからない
CAMERA_SUCCESS処理に成功し、正常終了

camera_result_t camera_manager_initialize ( int16_t  max_camera_count_,

)

カメラ管理システムのメモリを確保し、初期化する

Warning
カメラ管理システムはリニアアロケータを使用したメモリ確保を行うため、本APIの失敗時を含め、リソースの破棄はリニアアロケータの破棄によって行う
Parameters
[in]max_camera_count_カメラ管理システムに登録可能なカメラ数上限値
[in,out]allocator_リニアアロケータ
[out]out_camera_manager_カメラ管理システム格納先
Return values
CAMERA_INVALID_ARGUMENT以下のいずれか
  • allocator_ == NULL
  • out_camera_manager_ == NULL
  • *out_camera_manager_ != NULL
  • max_camera_count_ <= 0
  • メモリ割り当てアドレス異常
CAMERA_NO_MEMORYメモリ確保に失敗
CAMERA_SUCCESS処理に成功し、正常終了
