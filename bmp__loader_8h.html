Detailed Description

BMPファイルのロード処理を行うAPIの定義

Author
chocolate-pie24

以下のファイルをサポートする

  • 非圧縮BMPファイル
  • ピクセルのチャンネルカウントはRGB or RGBAのみ
  • 画像の高さがint16_tに収まること
  • 画像の幅がが0より大きく、かつint16_tに収まること

◆ bmp_loader_load()

resource_result_t bmp_loader_load ( const char *  fullpath_,
bmp_loader_t *  bmp_loader_ 
)

BMPファイルをロードする

Note
ロード後のピクセルデータは以下の特徴を持つ
  • BGR -> RGBの順でピクセルデータが格納される
  • 24bit(RGB)形式の場合でもpaddingは0で密にデータが格納される
  • 画像原点は左上を基準とする
処理に失敗した場合、bmp_loader_の状態は不変
Parameters
[in]fullpath_BMPファイルフルパス
[in,out]bmp_loader_ロードデータ格納先構造体インスタンスへのポインタ
Return values
RESOURCE_INVALID_ARGUMENT以下のいずれか
  • fullpath_ == NULL
  • bmp_loader_ == NULL
RESOURCE_BAD_OPERATION以下のいずれか
  • bmp_loader_->pixels != NULL
  • メモリシステム未初期化
RESOURCE_LIMIT_EXCEEDEDメモリシステムの使用可能範囲上限超過
RESOURCE_NO_MEMORYメモリ確保失敗
RESOURCE_FILE_OPEN_ERRORファイルオープン失敗
RESOURCE_FILE_CLOSE_ERRORファイルクローズ失敗
RESOURCE_UNDEFINED_ERROR未定義エラーが発生
RESOURCE_FILE_READ_ERRORヘッダまたはピクセル情報の読み込みに失敗
RESOURCE_UNSUPPORTED_FILEサポート対象外のBMPファイル(DEBUG_BUILD or TEST_BUILDで詳細なログが出力される)
RESOURCE_OVERFLOW計算過程でオーバーフロー発生
RESOURCE_DATA_CORRUPTEDピクセル読み込みサイズ異常またはヘッダ情報破損
RESOURCE_SUCCESS処理に成功し、正常終了
