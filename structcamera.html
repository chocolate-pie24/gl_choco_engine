Detailed Description

カメラ内部状態管理構造体

Note
カメラ姿勢に関しては以下を念頭に置くこと
  • 座標系: 右手座標系
  • Roll: Z軸回りの回転
  • Pitch: X軸回りの回転
  • Yaw: Y軸回りの回転
  • カメラ前方方向: Z軸マイナス方向
