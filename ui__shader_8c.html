◆ ui_shader_create()

renderer_result_t ui_shader_create ( const char *  file_path_,
const char *  name_,
renderer_backend_context_t *  backend_context_,
ui_shader_t **  out_ui_shader_ 
)

UIシェーダーリソースインスタンスのメモリを確保し初期化する

以下の処理を行う

  • out_ui_shader_自身のリソース確保
  • シェーダーソースのコンパイル
  • シェーダーモジュールのリンク
  • UIシェーダーが扱うプロジェクション行列のLocation取得
  • UIシェーダーが扱うビュー行列のLocation取得
  • UIシェーダーが扱うモデル行列のLocation取得
Parameters
file_path_シェーダーソース格納ファイルパス(文字列の最後を'/'にすること)
name_シェーダーソースファイル名称(拡張子は含まない)
backend_context_レンダラーバックエンドコンテキストへのポインタ
out_ui_shader_リソース確保対象UIシェーダーリソースへのダブルポインタ
Return values
RENDERER_INVALID_ARGUMENT以下のいずれか
  • file_path_ == NULL
  • name_ == NULL
  • backend_context_ == NULL
  • out_ui_shader_ == NULL
  • *out_ui_shader_ != NULL
RENDERER_NO_MEMORYメモリ確保失敗
RENDERER_LIMIT_EXCEEDEDメモリシステムのメモリ使用量範囲上限超過
RENDERER_RUNTIME_ERROR以下のいずれか
  • 計算過程でオーバーフロー発生(文字列長さ異常)
  • ユニフォーム変数のLocation取得に失敗
RENDERER_DATA_CORRUPTED内部データ破損が発生
RENDERER_BAD_OPERATION以下のいずれか
  • レンダラーバックエンドが未初期化
  • シェーダーソースが既にコンパイル済み
  • シェーダーモジュールが既にリンク済み
RENDERER_SHADER_COMPILE_ERROR以下のいずれか
  • シェーダーモジュールのGPU側リソース確保に失敗
  • シェーダーソースのコンパイルに失敗
RENDERER_SHADER_LINK_ERRORシェーダーモジュールのリンクに失敗
RENDERER_SUCCESS処理に成功し、正常終了

UIシェーダーリソースインスタンスが保持するリソースと、自身のメモリを開放する

Note
  • 本APIを使用した後、ui_shader_ == NULLとなり再使用不可
  • 2重デストロイ許可
  • GPU側のシェーダープログラムリソースも破棄される
Parameters
backend_context_Renderer Backendコンテキスト構造体インスタンスへのポインタ
ui_shader_破棄対象UIシェーダーリソースインスタンスへのダブルポインタ

Return values

RENDERER_BAD_OPERATION以下のいずれか
  • シェーダープログラムが未リンク状態
  • backend_context_が未初期化でshader_vtableがNULL
