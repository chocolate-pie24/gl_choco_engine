◆ renderer_texture_bind

pfn_renderer_texture_bind renderer_texture_vtable::renderer_texture_bind

テクスチャをactiveにし、bindする

Note
呼び出し側が保持している現在のtexture unit / texture handleキャッシュとtexture_handle_が保持するunit / handleと等しい場合はactive化 / bindを行わない
Parameters
[in]texture_handle_bind対象テクスチャハンドル保有構造体インスタンスへのポインタ
[in,out]out_texture_unit_現在activeになっているテクスチャユニット番号
[in,out]out_texture_internal_handle_現在Bindされているテクスチャハンドル
Return values
RENDERER_INVALID_ARGUMENT以下のいずれか
  • texture_handle_ == NULL
  • out_texture_unit_ == NULL
  • out_texture_internal_handle_ == NULL
RENDERER_DATA_CORRUPTED以下のいずれか
  • texture_handle_->handle == 0
  • texture_handle_->unit_number < 0
RENDERER_SUCCESS処理に成功し、正常終了

pfn_renderer_texture_create renderer_texture_vtable::renderer_texture_create

テクスチャGPU側リソース構造体インスタンスのメモリを確保し、テクスチャ設定を行い初期化する

Parameters
[in]unit_num_シェーダーが参照するテクスチャ用スロット番号
[in]min_filter_config_テクスチャ縮小表示の際の設定値
[in]mag_filter_config_テクスチャ拡大表示の際の設定値
[in]wrap_config_s_axis_テクスチャがラップする部分の表示設定値(s軸)
[in]wrap_config_t_axis_テクスチャがラップする部分の表示設定値(t軸)
[out]texture_handle_リソース確保、初期化対象テクスチャGPUリソース構造体インスタンスへのダブルポインタ
Return values
RENDERER_INVALID_ARGUMENT以下のいずれか
  • texture_handle_ == NULL
  • *texture_handle_ != NULL
  • min_filter_config_が規定値外
  • mag_filter_config_が規定値外
  • wrap_config_s_axis_が規定値外
  • wrap_config_t_axis_が規定値外
  • unit_num_ < 0
RENDERER_BAD_OPERATIONメモリシステム未初期化
RENDERER_LIMIT_EXCEEDEDメモリシステム使用可能範囲上限超過
RENDERER_NO_MEMORYメモリ確保失敗
RENDERER_SUCCESS処理に成功し、正常終了
